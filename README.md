# Caradel Attestation Maker 📝

**Caradel Attestation Maker** est un outil web simple, intuitif et professionnel conçu pour générer rapidement des attestations de travail standardisées pour l'entreprise **CARADEL SARL**.

## 🚀 Fonctionnalités

- **Aperçu en direct** : Visualisez les modifications en temps réel sur le document final.
- **Personnalisation complète** : Saisissez les informations de l'employé (Nom, Date de naissance, Poste, Dates de service).
- **Formatage intelligent** : Gestion automatique du genre (Monsieur/Madame) et mise en forme des listes de missions.
- **Export PDF** : Générez un document PDF prêt à l'emploi et à l'impression en un clic.
- **Identité visuelle** : Intégration du logo de l'entreprise et des informations légales (RC, NIU, etc.).
- **Interface responsive** : Utilisable sur ordinateur et mobile.

## 🛠️ Utilisation

1. Ouvrez le fichier `index.html` dans votre navigateur.
2. Remplissez le formulaire de personnalisation à gauche (ou en haut sur mobile).
3. Vérifiez le rendu dans la section de prévisualisation.
4. Cliquez sur **Générer PDF** pour sauvegarder le document.
